# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately.

**Do not create a public GitHub issue.** Instead, email the details to the repository owner.

## What to include

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

## Response

You can expect a response within 72 hours. We'll work with you to understand and address the issue.
